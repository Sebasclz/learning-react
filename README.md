# Aprendiendo React

Este repositorio contiene una colección de proyectos y ejercicios realizados para aprender y practicar React. Cada proyecto aborda diferentes aspectos del desarrollo de aplicaciones utilizando React como biblioteca principal para la creación de interfaces de usuario.

## Proyectos

### 1. Twitter Follow Card

Un componente simple de tarjeta que simula la funcionalidad de "seguir" y "dejar de seguir" en Twitter. Este proyecto se enfoca en el uso de componentes, estado y eventos.

### 2. Tic Tac Toe

Una implementación del clásico juego Tic Tac Toe, donde se practican conceptos como el manejo del estado, renderizado condicional y la lógica de juego en React.

### 3. Mouse Follower

Un pequeño proyecto donde un elemento en pantalla sigue el movimiento del ratón, utilizando eventos y efectos en React para capturar y responder a la interacción del usuario.

### 4. Prueba Técnica: Fetching de Datos

Un ejercicio técnico para aprender a hacer solicitudes de datos (fetching) en React, mostrando cómo manejar el estado de carga, errores y la renderización de datos dinámicos.

### 5. Prueba Técnica: Buscador de Películas

Un proyecto que implementa un buscador de películas utilizando una API externa. Se enfoca en el manejo de formularios, peticiones HTTP y la renderización dinámica de resultados.

### 6. Carrito de la Compra

Una aplicación de carrito de compras que permite agregar y eliminar productos, así como aplicar filtros. Este proyecto integra conceptos de estado global, manipulación de listas y gestión de eventos en React.

### 7. React Router

Un proyecto que demuestra cómo implementar la navegación entre diferentes páginas utilizando React Router, manejando rutas y subrutas en una aplicación React.

## Instalación y Uso

1. Clona este repositorio:

   ```bash
   git clone https://github.com/tu-usuario/nombre-del-repositorio.git
   ```

2. Navega a la carpeta del proyecto:

   ```bash
   cd nombre-del-repositorio
   ```

3. Instala las dependencias del proyecto:

   ```bash
   npm install
   ```

4. Ejecuta el proyecto:

   ```bash
   npm run dev
   ```

5. Abre tu navegador y visita `http://localhost:3000` para ver el proyecto.
